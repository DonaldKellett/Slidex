Slidex

A custom presentation tool written purely in HTML5, CSS3 and Javascript

Proudly presented to you by Donald Leung

What is Slidex?

Slidex is the best alternative to commonly used presentation-making software such as Microsoft's Microsoft Powerpoint and Google Drive's Presentation tool. Features include:

  • High Flexibility
  • High Customizability
  • User Friendly
  • Easy Navigation
  • Developer Friendly

... and a lot, lot more.

The fact that Slidex is built purely on HTML5, CSS3 and Javascript means that it is customizable to the core. Want images with pretty, colored borders? No problem! Just delve deep into the CSS and set the image's border property to something like border: 5px solid #5a5a5a, and ... ta da! You've just created beautiful borders around the image!

It also means that you can do a lot more on Slidex than you can do on Powerpoints, Google Presentations or even Prezi. For example, on Presentations like Microsoft Powerpoint or Prezi, it is next to impossible to place a button on your presentation. However, since Slidex is purely HTML5, CSS3 and Javascript, you can easily customize your own button and even how it behaves!
